# Google ADK Learning

This repository contains code and experiments for learning the Google ADK (Application Development Kit).

## Contents

- `session_management/` — examples for building stateful agent sessions
- `qa_agent/` — question-answering agent implementations
- `tool_agent/` — tools and agent workflows for extended capabilities
- `first_agent/` — starter agent examples

## Notes

- Do not commit sensitive files such as `.env` or API keys.
- A `.gitignore` file is included to protect environment files and common Python artifacts.

## Purpose

This folder is used to store learning exercises, sample scripts, and notes on how to build applications with Google ADK and agent-based workflows.
